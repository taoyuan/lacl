> An ACL component for loopback based on role, user and resource


## Install

```
$ npm install --save loopback-component-acls
```


## Usage

```js
const loopbackComponentAcl = require('loopback-component-acls');

loopbackComponentAcl('unicorns');
//=> 'unicorns & rainbows'
```
